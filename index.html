Sélection d'images — carrelage, murs, accents

Chaque image est marquée des versions concernées (cf. VERSIONS.md) et d'un thème. Filtrées selon les contraintes fermes : sols clairs en carrelage (pas de parquet), finition satinée, murs blancs/sable, lumière chaude, accents doux (accents profonds tolérés pour V1 uniquement). Sur quelques photos le sol est masqué par un tapis — c'est précisé dans la note. Cliquer « Ouvrir sur Pinterest » pour épingler.
Mode d'emploi : touchez le ♥ sur les images qui vous plaisent, puis envoyez votre sélection via le bouton WhatsApp qui apparaît en haut. Vos favoris restent enregistrés sur votre appareil.
